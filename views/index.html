안녕하세요! Node.js를 활용해 만든 API를 테스트하는 서버입니다.

아래 API 명세서롤 참조하시면서 테스트를 진행해보세요. 깃허브 리포지토리 보러가기

POSTMAN 을 설치하시면 더 간단하게 API를 사용해보실 수 있습니다. POSTMAN 설치 페이지

  • 기능

    API URL

    Method

    Request(가져 갈 데이터)

    Response(서버로부터 받아올 데이터)

    Response(error)

  • 게시글 작성

    http://limportfolio.shop/community/free/write

    POST

    {
    "user": "작성자ID",
    "password": "작성자PW",
    "title": "글 제목",
    "content": "글 내용"
    }

    {
    "success": true
    }

  • 게시글 검색

    http://limportfolio.shop/community/free/list

    GET

    http://limportfolio.shop/community/free/list?search="검색어"

    "검색어" 안에 검색 할 제목/내용 입력

    {
    "success": true
    "posts": [글 정보(게시글 ID 포함)]
    }

  • 게시글 상세 조회

    http://limportfolio.shop/community/free/view/:postId

    GET

    http://limportfolio.shop/community/free/view/:postId

    "view/검색 할 게시글 ID" 형식으로 요청

    {
    "success": true,
    "posts": [글 정보(게시글 ID 포함)]
    }

  • 게시글 수정

    http://limportfolio.shop/community/free/write

    PUT

    {
    "postId": "게시글 ID(Object Id)",
    "password": "작성자 PW",
    "title": "글 제목",
    "content": "글 내용"
    }

    {
    "success": true
    }

    {
    "success": false,
    "errorMessage": "해당 게시글에 권한이 없거나 비밀번호가 일치하지 않습니다."
    }

  • 게시글 삭제

    http://limportfolio.shop/community/free/delete

    DELETE

  • 댓글 조회(답글 포함)

    http://limportfolio.shop/community/comment/view/:postId

    GET

    http://limportfolio.shop/community/comment/view/:postId

    "view/검색 할 게시글 ID" 형식으로 요청

    {
     "success": true,
     "searchComments": [
       {
        댓글 정보(댓글 ID 포함)
        "replyList": [답글 리스트 정보(답글 ID 포함)]
       },
     ]
    }
